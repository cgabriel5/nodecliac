Overview

Bash tab-completion is awesome but adding it to CLI programs can be a pain. nodecliac simplifies this. The idea is to map the program's commands and flags in an .acmap file and well, that's it. nodecliac handles the rest. The end result? Your program having Bash tab-completion.

Notes:
1. nodecliac was originally made in mind for NodeJs CLI programs which explains the name. However, as development continued it proved useful for non NodeJS programs as well.
2. Bash and Perl are used in tandem to provide tab-completion. nodecliac's tools (parser, formatter, etc.) are written inJavaScript and therefore require NodeJS.
Example: Sublime Text Tab-completion

As an example, let's add Bash tab-completion to Sublime Text's subl command.


1.
Create subl.acmap with the following contents: 
subl = [
	--project
	--command
	--new-window?
	--add
	--wait?
	--background?
	--help?
	--version?
]
2.
Let nodecliac take over.
$ nodecliac make --source=path/to/subl.acmap --add
3.
Finally, open a new Terminal and start using!
