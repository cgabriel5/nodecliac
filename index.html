What is nodecliac?

bash-completion for CLI programs is awesome. It enhances the user experience by completing filenames, commands, flags, etc. What isn't is the time and effort required to add it.


nodecliac provides an easier way. By simply mapping a program's commands with their flags in an .acmap file, nodecliac can leave a program Bash completion ready.


The goal behind this project is two-fold: minimize efforts needed when adding bash-completion and build a collection of community made completion packages for all to enjoy. This way, more programs come baked with bash-completion support.

Install 
$ sudo curl -Ls https://raw.githubusercontent.com/cgabriel5/nodecliac/master/install | bash -s && source ~/.bashrc
