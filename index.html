Easy Bash tab-completion for CLI programs
Overview

Bash tab-completion is awesome but adding it to CLI apps can be a pain. nodecliac aims to simply this. With nodecliac, the idea is to map the program's commands and flags in an .acmap file and well, that's it. nodecliac handles the heavy lifting and your CLI app now has Bash tab-completion.

Notes:
1. nodecliac was originally made in mind for NodeJs CLI applications which explains the name. However, as development continued it proved useful for non NodeJS applications as well.
2. Currently, Bash and Perl are used for tab-completion. The parser (converts .acmap files to .acdef files), formatter, and other CLI tools are made using NodeJS/JavaScript.
Getting Started
1.
Install nodecliac via yarn, npm, or manually.
2.
Create the CLI app's command.acmap file.
3.
Pass command.acmap to nodecliac and let it handle the rest!
4.
Open a new Terminal and start using!
Example:

As an example, let's add Bash tab-completion to Sublime Text's subl command.

1.
Create subl.acmap with the following contents: 
subl = [
	--project
	--command
	--new-window?
	--add
	--wait?
	--background?
	--help?
	--version?
]
2.
Let nodecliac take over.
$ nodecliac make --source=path/to/subl.acmap --add
3.
Finally, open a new Terminal and start using!
4.
See docs for complete details.
